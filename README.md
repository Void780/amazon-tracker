# Amazon Price Tracker Bot

Telegram bot that monitors Amazon product prices and sends alerts when they drop.

- Search the bot on Telegram
- Send an Amazon product link
- Bot reads and stores the current price
- Sends a Telegram notification when price drops

## How to use

Search **@AmazonVoid_bot** on Telegram or click the link below to start.

[Start the bot](https://t.me/AmazonVoid_bot)

## Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot |
| `/lista` | View and remove tracked products |
| Amazon link | Add a product to tracking |

## Running the Bot

Send any Amazon product link directly in chat and the bot will start tracking it automatically.
